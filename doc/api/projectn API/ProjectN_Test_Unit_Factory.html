static void get( $model , [array $data = null] , [boolean $autoCreateRelatedObjects = true] )

Creates a new $model object to the Sqlite database with minimum required fields. You can pass an array of data to override default values.

If not Vendor or Category is available, a default one will be created so that the Poi can link to it. This is necessary for the Poi to be saved. This feature can be set to false in the second parameter
