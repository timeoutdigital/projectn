Class geoEncode

Gets geo data using Google.

Author(s):
  • Tim Bowler <timbowler@timeout.com>
Version:  

1.0.0

Example of usage:

The addressString must contain as much address info as possible

  1.   $geoEncode = new geoEncode();
  2.   $geoEncode->setAddress( $addressString );
  3.  
  4.   //Set longitude and latitude
  5.   $poiObj[ 'longitude' ] = $geoEncode->getLongitude();
  6.   $poiObj[ 'latitude' ]  = $geoEncode->getLatitude();

If a geocode cannot be found then an Exception is thrown
