Class curlImporter

Retrieve an XML feed from a website as Simple XML

Author(s):
  • Tim Bowler <timbowler@timout.com>

1.0.1

Example:

  1.   $xmlObj = new curlImporter
  2.  
  3.   $parameters = array('from' => '2010-01-01', 'to' => '2010-01-30');
  4.   $returnObj = $xmlObj->pullXml('http://www.timeout.pt/', 'xmllist.asp', $parameters);
