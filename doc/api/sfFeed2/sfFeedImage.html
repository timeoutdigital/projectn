Class sfFeedImage

Adds support for icons/images for atom and rss feeds in an unified extensible way.

Both (rss and atom) specs are quite limited, but this class should cope with possible enhancements. Note that not everything is used. Atom at the moment only takes favicon (1:1 ratio) and image (approx 2:1 ratio) image urls. Rss only takes the image, but expects additionally the size of that image.

Author(s):
